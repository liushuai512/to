js的组成？ js由三部分组成，ECMAScript、DOM和BOM ***** ECMAScript：是规定了js中的一些基本规范、基本操作语句、数据类型和关键字与保留字等等，相当于js的API ECMAScript的版本：ES5 ES6 ES2016+ DOM：文档对象模型（Document Object Model） 整个html文件就是一个文档 里面的元素关系组成了一个树形结构（一个根源引出多个分支） js用来操作这些DOM元素（js可以操作html中的所有元素） --标签 --元素 DOM的顶层对象是document； BOM：浏览器对象模型(Browser Object Model) BOM可以让js直接和浏览器的进行对话； BOM由多个对象组成，其中代表浏览器窗口的Window对象是BOM的顶层对象，其他对象都是该对象的子对象；
