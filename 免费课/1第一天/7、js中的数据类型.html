js中的数据类型分为基本数据类型和引用数据类型 基本数据类型： 数字(number) 布尔(boolean) 字符串(string) null undefined 引用数据类型： 对象数据类型（object）： Object （对象类） Array（数组类） 正则类（RegExp） Math（数学函数类） Data（时间类） String（字符串类） Number（数字类） 函数数据类型（function） js是基于对象的脚本语言 object 在js中是一个泛指，任何的内容在js中都是一个对象（object），我们为了更好的管理这些对象将他们分成各种类，那么每一个具体的对象就是这个类中的实例 大 --> 中 --> 小 对象-->类-->实例 菜 -->八大菜系-->宫保鸡丁
